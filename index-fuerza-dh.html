Sí. En tu caso, el tamaño se controla casi todo desde el CSS.

El motivo de que ahora se vean más pequeños en PC suele ser una combinación de estas dos cosas que ya se ven en tus archivos:

  • los vídeos están dentro de grids de 2, 3 o más columnas en páginas como video-grid-2x2, video-grid-2x3 o video-grid-1x3

  • el iframe tiene width: 100%, así que su tamaño depende totalmente del ancho de la columna donde cae

Qué tocar

Opción 1. La más sencilla: aumentar el ancho útil de la página

Si en el CSS nuevo tienes algo tipo .main, .container o similar con un max-width, súbelo.

Por ejemplo, si tienes algo así:

CSS
.main {
    max-width: 1100px;
    margin: 0 auto;
    padding: 20px;
}

prueba con:

CSS
.main {
    max-width: 1500px;
    margin: 0 auto;
    padding: 20px 24px;
}

Eso hará que cada vídeo tenga más anchura en PC.

Opción 2. Forzar menos columnas en escritorio

Si quieres que se vean claramente más grandes, lo más efectivo es pasar de 3 columnas a 2 en muchas páginas.

Por ejemplo, en vez de algo visualmente equivalente a 3 columnas:

CSS
.video-grid {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 24px;
}

usa:

CSS
.video-grid {
    display: grid;
    grid-template-columns: repeat(2, minmax(0, 1fr));
    gap: 24px;
}

Con eso cada vídeo crece bastante.

Opción 3. Responsive mejor ajustado para PC y móvil

Esta es la que te recomiendo. Deja una sola clase genérica y controla columnas por ancho de pantalla:

CSS
.video-grid {
    display: grid;
    grid-template-columns: repeat(2, minmax(0, 1fr));
    gap: 24px;
    width: 100%;
}

.video-container iframe {
    width: 100%;
    aspect-ratio: 16 / 9;
    height: auto;
    border: 0;
}

@media (max-width: 900px) {
    .video-grid {
        grid-template-columns: 1fr;
    }
}

Resultado

  • PC: 2 vídeos por fila, bastante grandes

  • móvil: 1 por fila, bien visible

Qué no tocaría

}

El problema no está ahí, sino en el ancho que recibe el iframe desde el grid.

Resumen

Haz estas dos pruebas, en este orden:

CambioEfecto
--container: 1600pxmás ancho total de página
minmax(520px, 1fr)vídeos bastante más grandes

Si quieres un cambio directo y seguro, deja esto:

CSS
--container: 1600px;
CSS
.video-grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(520px, 1fr));
    gap: 24px;
    align-items: start;
}

Qué hacer con las notas técnicas

  </div>
</div>
